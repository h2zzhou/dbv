Browser settings

For debugging puposes, here are the settings to allow local file access in main browsers:

  • Mozilla Firefox: in the about:config page, set security.fileuri.strict_origin_policy to false.
  • Google Chrome: launch it from the command line with '--disable-web-security'.
  • Apple Safari: in Safari preferences, Advanced, tick 'Show Develop menu in menu bar' Then in the develop menu, tick: 'Enable webGL' and 'Disable Local File Restrictions'..

Mesh import

The mesh files need to contain normals to be loaded with XTK. Normals can be generated with Paraview:

  1. Load the shape,
  2. Paraview > Filters > 'Generate Surface Normals' (with or without 'Flip Normals')
  3. Set the 'Feature Angle' to 0,
  4. Apply,
  5. Save using vtk ASCII format
